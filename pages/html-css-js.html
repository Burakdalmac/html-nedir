HTML, CSS ve JavaScript Entegrasyonu

Bu sayfada HTML, CSS ve JS’in nasıl birlikte çalıştığını basit bir örnekle göreceksiniz.

Hoş Geldiniz!

Alttaki butona tıklayarak kartın stilini değiştirebilirsiniz.


<!-- HTML -->
<div class="kart" id="entegreKart">
  <h2>Hoş Geldiniz!</h2>
  <p>Alttaki butona tıklayarak stil değiştir.</p>
  <button id="stilDegistirBtn">Stili Değiştir</button>
</div>

<!-- CSS (style.css) -->
.kart {
  border: 2px solid #333;
  padding: 16px;
  border-radius: 8px;
  transition: background-color 0.3s;
}

<!-- JS (myscript.js) -->
document.getElementById("stilDegistirBtn").addEventListener("click", function() {
  document.getElementById("entegreKart").style.backgroundColor = "#e0f7fa";
});

Butona tıkladığında JavaScript, CSS stilini dinamik olarak değiştirir.
