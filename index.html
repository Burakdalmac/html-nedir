HTML Nedir?

HTML (HyperText Markup Language), web sayfalarını oluşturmak için kullanılan temel işaretleme dilidir. HTML, tarayıcılar tarafından okunarak internet sitelerinin yapılandırılmasını sağlar.

📜 HTML'nin Tarihçesi

HTML ilk kez **1991 yılında Tim Berners-Lee** tarafından geliştirilmiştir. O dönemde internet oldukça basitti ve sadece metin tabanlı içeriklerden oluşuyordu. Ancak HTML sayesinde metinler **bağlantılarla (hyperlinks) birbirine bağlanabilir** hale geldi.

Günümüzde HTML, gelişerek CSS ve JavaScript ile birlikte **modern web sitelerinin temel taşı** haline gelmiştir.

⚙️ HTML Nasıl Çalışır?

HTML, tarayıcılar tarafından yorumlanarak görsel bir web sayfasına dönüştürülür. Bir HTML sayfası, metinleri biçimlendirmek, resimler eklemek, bağlantılar oluşturmak ve çok daha fazlasını yapmak için çeşitli etiketlerden oluşur.

Örneğin, bir HTML dosyası şu şekilde başlar:

<!DOCTYPE html>
<html>
    <head>
        <title>Merhaba Dünya!</title>
    </head>
    <body>
        <h1>Merhaba Dünya!</h1>
        <p>Bu, basit bir HTML sayfasıdır.</p>
    </body>
</html>

🔑 HTML’nin Temel Yapısı

HTML, **etiketler (tags)** kullanılarak yapılandırılır. İşte en sık kullanılan HTML etiketleri ve açıklamaları:

  • <h1> - <h6>: Başlık etiketleridir. **<h1>** en büyük başlıktır, **<h6>** en küçüğüdür.
  • <p>: Paragraf eklemek için kullanılır.
  • <a href="URL">: Bağlantı (link) oluşturur.
  • <img src="resim.jpg" alt="Açıklama">: Sayfaya resim ekler.
  • <ul> ve <li>: Sırasız liste oluşturur.
