HTML Nedir?

HTML (HyperText Markup Language), web sayfalarını oluşturmak için kullanılan temel işaretleme dilidir. İnternetin temel yapı taşlarından biri olan HTML, tarayıcılar tarafından okunarak görsel bir web sayfasına dönüştürülür.

📜 HTML’nin Tarihçesi

HTML, 1991 yılında **Tim Berners-Lee** tarafından geliştirilmiştir. İnternetin ilk zamanlarında, web sayfaları sadece metin içeriyordu ve bağlantılar (hyperlinks) kullanılarak sayfalar birbirine bağlanabiliyordu. HTML'in ortaya çıkışıyla, web sayfaları görsel olarak daha düzenli ve anlaşılır hale geldi. Günümüzde ise HTML, web tasarımının temel bileşeni olmuştur.

⚙️ HTML'nin Temel Yapısı

HTML, etiketler (tags) kullanarak web sayfalarının yapısını oluşturur. Her HTML sayfası temel olarak şu öğelerden oluşur:

  • <!DOCTYPE html> : HTML5 belgesinin başlatıldığını belirten bir etikettir.
  • <html> : HTML belgesinin başlangıcını işaret eder.
  • <head> : Sayfa hakkında meta verileri, stil dosyaları, başlık gibi unsurlar bu etiket içinde yer alır.
  • <body> : Web sayfasının içerik kısmını barındırır. Sayfadaki metin, resim, bağlantı gibi tüm içerik buradadır.

HTML Belgelerinin Temel Yapısı:

<!DOCTYPE html>
<html lang="tr">
    <head>
        <meta charset="UTF-8">
        <title>HTML Nedir?</title>
    </head>
    <body>
        <h1>HTML Nedir?</h1>
        <p>HTML, web sayfalarını oluşturmak için kullanılan temel işaretleme dilidir.</p>
    </body>
</html>

🔑 HTML Etiketleri ve Özellikleri

HTML, içeriği biçimlendirmek ve sayfada düzen oluşturmak için çeşitli etiketler kullanır. İşte bazı temel HTML etiketleri:

  • <p>: Paragrafları belirtir.
  • <a>: Bağlantı oluşturur.
  • <h1> - <h6>: Başlık seviyelerini belirtir.
  • <img>: Sayfaya görsel ekler.
  • <ul> ve <li>: Sırasız liste oluşturur.

🌍 HTML Nerelerde Kullanılır?

HTML, yalnızca web sayfalarının yapısını belirlemekle kalmaz, aynı zamanda web sitelerinin temel tasarımına da yön verir. HTML, şu alanlarda yaygın olarak kullanılır:

  • Web Sayfaları: İnternette gezinirken karşılaştığınız hemen her sayfa HTML ile yapılmıştır.
  • Mobil Uygulama Tasarımı: Bazı mobil uygulamalar da HTML kullanılarak tasarlanabilir.
  • Web Uygulamaları: Çeşitli internet uygulamaları HTML ile yapılmaktadır.

🔗 HTML ile Etkileşimli Öğeler

HTML, etkileşimli öğeler eklemek için de kullanılır. Örneğin:

  • Formlar: Kullanıcılardan veri toplamak için HTML form etiketleri kullanılır.
  • Bağlantılar: Web sayfalarına yönlendiren linkler oluşturulabilir.
  • Tablolar: Verileri düzenli bir şekilde sunmak için tablolar kullanılır.

🔄 HTML ve Diğer Teknolojiler

HTML, sadece metin ve yapıyı tanımlar. Ancak modern web tasarımı için HTML, diğer teknolojilerle birlikte kullanılır:

  • CSS (Cascading Style Sheets): Sayfanın görünümünü düzenler. HTML ile sayfa yapısını belirlerken, CSS ile görsel düzen sağlanır.
  • JavaScript: Sayfalara etkileşim ve dinamik özellikler ekler. Örneğin, kullanıcı tıklamalarıyla bir şeylerin değişmesi gibi.

🖼️ HTML Örneği

İşte basit bir HTML sayfası örneği:

<html>
    <head>
        <title>Merhaba Dünya</title>
    </head>
    <body>
        <h1>Merhaba, bu bir HTML sayfasıdır!</h1>
        <p>HTML ile ilgili temel bilgileri bu sayfada öğrenebilirsiniz.</p>
    </body>
</html>

Sonuç

HTML, web geliştirme sürecinin temel taşıdır ve her web tasarımcı ve geliştiricisinin bilmesi gereken bir teknolojidir. HTML kullanarak web sayfalarının yapısını ve içeriğini belirleyebilir, CSS ve JavaScript ile sayfanın görünümünü ve işlevselliğini geliştirebilirsiniz.
